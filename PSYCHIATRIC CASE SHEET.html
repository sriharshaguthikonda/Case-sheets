    DRUG HISTORY:
    • Anticholinergic Medications (e.g., Diphenhydramine, Benztropine, Oxybutynin, TCAs):
      • Memory impairment
      • Confusion / Delirium
      • Agitation
      • Exacerbation of dementia
      • Hallucinations (visual more common)
    • Dopamine Agonists (e.g., Levodopa, Pramipexole, Ropinirole - for Parkinson's, RLS):
      • Hallucinations (often visual)
      • Confusion, Agitation
      • Impulse control disorders (e.g., gambling, hypersexuality)
      • Psychosis
      • Delusions
      • Mania / Hypomania
    • Corticosteroids (e.g., Prednisone, Dexamethasone):
      • Insomnia
      • Anxiety, Irritability, Agitation
      • Mania / Hypomania ("steroid-induced mania")
      • Depression
      • Cognitive changes (e.g., memory impairment, confusion, "steroid dementia")
      • Psychosis ("steroid-induced psychosis")
    • Beta-Blockers (e.g., Propranolol, Metoprolol):
      • Fatigue, Lethargy
      • Sleep disturbances, Nightmares
      • Depression
      • Confusion
      • Hallucinations
    • Opioids (e.g., Morphine, Oxycodone, Codeine):
      • Sedation, Confusion
      • Dysphoria or Euphoria (variable)
      • Delirium
      • Depression (with chronic use/dependence/withdrawal)
      • Anxiety (prominent during withdrawal)
      • Hallucinations
    • Benzodiazepines & Z-Drugs (e.g., Diazepam, Lorazepam, Zolpidem):
      • Anterograde amnesia
      • Depression (can unmask or worsen)
      • Withdrawal symptoms: Anxiety, Insomnia
      • Paradoxical excitement/agitation
      • Delirium
      • Withdrawal symptoms: Seizures, Psychosis
    • Stimulants (e.g., Methylphenidate, Amphetamine - for ADHD, Narcolepsy):
      • Insomnia
      • Anxiety, Agitation, Irritability
      • Obsessive-compulsive symptoms (can be exacerbated)
      • Tics (can be exacerbated or induced)
      • Psychosis / Mania
    • Interferons (e.g., Interferon-alpha - for Hepatitis C, some cancers):
      • Fatigue
      • Depression
      • Anxiety, Irritability
      • Insomnia
      • Cognitive impairment ("brain fog")
      • Suicidal ideation
      • Psychosis
      • Mania / Hypomania
      • Delirium
    • Certain Antibiotics:
      • Fluoroquinolones (e.g., Ciprofloxacin, Levofloxacin):

    3. Treatment Plan:
      • Pharmacotherapy:
        • Specific medication(s) considered (class, name, starting dose, titration schedule).

        • (Examples: Antidepressants - SSRIs, SNRIs, TCAs, MAOIs; Antipsychotics - typical/atypical; Mood stabilizers - Lithium, Valproate, Lamotrigine, Carbamazepine; Anxiolytics - Benzodiazepines, Buspirone; Hypnotics; Stimulants for ADHD)
      • Psychotherapy:

        • Rationale, goals, expected duration/frequency.
        • Referral if needed.
      • Social/Environmental Interventions:
        • Case management, vocational rehabilitation, housing support.
        • Family psychoeducation and support.
        • Linkage to community resources and support groups.
        • Lifestyle modifications (sleep hygiene, diet, exercise, stress management).
      • Admission required: Yes/No (Indication as above).
    4. Follow Up:
      • Timing of next appointment (crucial for safety and monitoring if starting new medication).
      • Frequency of subsequent appointments.
      • Monitoring plan (symptoms, side effects, adherence, functioning).

      • Collaboration with other healthcare providers (e.g., GP, therapist).
