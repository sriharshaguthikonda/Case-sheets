Lymph Node Swelling

1ST COMPLAINT: NECK SWELLING/LUMP

  • S: Site "Where exactly is the lump? Can you point to it?"
    • ANTERIOR TRIANGLE
      • Thyroid nodule
      • Thyroglossal cyst
      • Lymphadenopathy
    • POSTERIOR TRIANGLE
      • Lymphadenopathy
      • Branchial cyst
      • Cystic hygroma
    • MIDLINE
      • Thyroglossal cyst
      • Dermoid cyst
      • Goiter
    • SUBMANDIBULAR
      • Salivary gland stone/tumor
      • Lymphadenopathy
  • O: Onset "When did you first notice the lump? Did it appear suddenly or grow over time?"
    • SUDDEN
      • Reactive lymph node
      • Infected cyst
      • Hematoma
    • GRADUAL
      • Goiter
      • Tumor (benign/malignant)
      • Chronic lymphadenopathy
  • D: Duration & Painlessness "How long has the lump been there? And is it painful?"
    • SHORT DURATION + PAIN
      • Inflammatory
    • LONGER DURATION + PAINLESS
      • Malignant
    • LONG DURATION + LESS PAIN
      • Chronic Inflammation
    • LONG DURATION + PAINLESS
      • Benign
  • A: Aggravating Factors "Does anything seem to make the lump larger or more painful?"
  • R: Relieving Factors "Is there anything that makes it smaller or less uncomfortable?"
  • C: Character "What does the lump feel like? Is it soft, firm, or hard? Is it mobile?"
    • SOFT/FLUCTUANT
      • Cyst
      • Thyroglossal cyst
      • Branchial cyst
    • FIRM
      • Thyroid nodule
      • Lymphadenopathy
    • HARD
      • Malignancy
      • Calcified node
    • MOBILITY: Is it attached to the skin or underlying structures?
  • R: Radiation "Do you feel any pain or sensation spreading from the lump?"
  • A: Associated Symptoms "Have you noticed any other symptoms along with the neck lump?"
    • VOICE CHANGES (Hoarseness)
      • Thyroid cancer
      • Laryngeal issue
    • DIFFICULTY SWALLOWING (Dysphagia)
      • Goiter
      • Pharyngeal pouch
      • Esophageal issue
    • DIFFICULTY IN RESPIRATION
      • Airway compression
    • WEIGHT CHANGES
      • Hyper/Hypothyroidism
      • Malignancy
    • FEVER, NIGHT SWEATS
      • Lymphoma
      • Tuberculosis
      • Systemic infection
    • RECENT INFECTION (e.g., sore throat)
      • Reactive lymphadenopathy
    • OTHER SWELLINGS
      • Neurofibromatosis
      • Hodgkin's disease
    • SECONDARY CHANGES: Softening, Ulceration, Fungation, Inflammation
    • IMPAIRMENT OF FUNCTION: e.g., neck movement
  • P: Progression "Has the lump changed in size since you first noticed it?"
    • VERY RAPID
      • Inflammatory
    • RAPIDLY
      • Malignant
    • SLOWLY
      • Benign
    • REGRESSIVE
      • Inflammatory
    • STATIC
      • Benign
    • STATIC -> RAPID
      • Benign -> Malignant transformation
  • P: Pattern "Does the size of the lump change? Does it come and go?"
  • E: Effect on Life "How has this lump affected you? Is it causing any concern, discomfort or disfigurement?"
  • R: Red Flags "I need to ask a few specific questions to make sure we're not missing anything important."
    • HARD, FIXED, NON-TENDER LUMP
      • Malignancy
    • PERSISTENT HOARSENESS
      • Laryngeal cancer
      • Recurrent laryngeal nerve palsy
    • WEIGHT LOSS, FEVER, NIGHT SWEATS
      • Lymphoma
      • Metastatic cancer
      • TB
      • AIDS
    • ULCERATION OR SKIN CHANGES OVER THE LUMP
      • Malignancy
    • STRIDOR (noisy breathing)
      • Airway compression - medical emergency!
  • S: Severity / Scale "On a scale of 1 to 10, how much is this lump bothering you?"
